modern.ie.japanizer
===================

scripts to set up Japanese environments in modern.IE virtual machines.

Compile japanize-modern-ie.au3 with [AutoIt](http://www.autoitscript.com/site/autoit/) compiler and run

```
$ ./start_and_japanize_vm.sh
```
